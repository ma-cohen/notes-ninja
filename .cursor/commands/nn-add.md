# Add Notes/Ideas

Add notes, ideas, or learnings to the knowledge base in the `notes/` folder.

## Instructions

1. List all `.md` files in `notes/` folder
2. Read frontmatter (title, tags) from each file to understand topics
3. Decide: append to existing file OR create new file
   - **Append** if content clearly fits an existing file's topic
   - **Create new** if no good match exists
4. When creating new files:
   - Use descriptive kebab-case names (e.g., `python-tips.md`, `app-ideas.md`)
   - Add frontmatter with title, created date, and relevant tags
5. Add content as a dated entry using today's date

## Note File Format

```markdown
---
title: Topic Name
created: YYYY-MM-DD
tags: [tag1, tag2]
---

## Entry - YYYY-MM-DD
Content here...
```
