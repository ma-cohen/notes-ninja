# Query Notes

Search and query the notes and todos in this repository.

## Instructions

1. Search through all files in `notes/` and `todos/`
2. Find relevant content matching the user's query
3. Summarize findings in a helpful response
4. Cite which files the information came from

## Guidelines

- Keep responses concise
- Provide summaries, not full file dumps
- Group related information together
- Always mention the source file for each piece of information
